GPA Calculator

Enter a comma separated list of grades.

Display input box for user to enter grades along with a button to kick off the calculation. Add an event listener to the button that will do the following when clicked: Get the string of grades from the input Convert the string to an array (String.split(',')), clean up any extra spaces, and make the grades all uppercase. Do a lookup on each grade to convert it to it's point value (ie A = 4.0) Total up all the point values, and divide by the number of grades to get the GPA Output the GPA to the browser
